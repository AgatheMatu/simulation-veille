# SimuVeille

Simulation pédagogique de fil social fictif pour travailler la veille informationnelle et la détection d'attaques informationnelles.

## Mise en ligne avec GitHub Pages

1. Créer un nouveau dépôt GitHub, par exemple `simulation-veille`.
2. Envoyer tous les fichiers de ce dossier à la racine du dépôt :
   - `index.html`
   - `styles.css`
   - `app.js`
   - `.nojekyll`
   - les fichiers `scenario-*.json`
3. Ouvrir `Settings`, puis `Pages`.
4. Dans `Build and deployment`, choisir :
   - Source : `Deploy from a branch`
   - Branch : `main`
   - Folder : `/root`
5. Enregistrer.

Après quelques minutes, la page sera disponible à une adresse du type :

`https://VOTRE-NOM.github.io/simulation-veille/`

## Modifier les scénarios

Les scénarios peuvent être modifiés directement dans le mode enseignant de la page, puis exportés en JSON.
